# Natural-Language-Processing-NLP-
A Natural Language Processing (NLP) project comparing TF-IDF and Word2Vec (Skip-gram) representations across Machine Learning and Deep Learning models (RNN, LSTM, GRU) for 10-category multi-class text classification
This repository contains a Natural Language Processing (NLP) study on multi-class text classification using a large-scale, 10-category question-answer dataset. The project provides a comparative analysis of different text representation techniques and modeling architectures to determine the most effective approach for categorizing text.  Specifically, the project evaluates sparse TF-IDF representations paired with baseline machine learning models, such as Logistic Regression, Naive Bayes, and Random Forest. These are contrasted against dense Skip-gram (Word2Vec) embeddings applied to deep sequence models, including SimpleRNN, GRU, LSTM, and their bidirectional variants. The experiments demonstrate that neural architectures utilizing dense embeddings are better equipped to capture semantic context and long-range dependencies, with the Skip-gram + Bidirectional GRU model achieving the best overall classification accuracy and macro-averaged F1-score.
